getMoveDirection(board, start, current, next) → {number|string}

Flood Fill Algorithm for Area Calculation This function calculates the safe area around a given position on any board size. It's used to: 1. Evaluate path safety on different map sizes 2. Avoid getting trapped in small areas 3. Make decisions based on available space The algorithm works by: - Starting from a given position - Exploring all reachable cells in the current board dimensions - Counting cells that aren't blocked by snake bodies - Returning the total safe area size
